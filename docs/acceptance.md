# 交付验收清单

- [ ] 使用 OpenAI SDK 仅修改 base_url 和 api_key 可调用已配置模型。
- [ ] 可创建租户、项目、应用。
- [ ] 可创建供应商、模型、部署、价格。
- [ ] 供应商密钥加密保存，不明文展示。
- [ ] 可申请、审批、生成、禁用 API Key。
- [ ] Key 明文只展示一次，数据库仅保存 hash。
- [ ] Key 可设置模型范围、预算、RPM/TPM/QPS。
- [ ] 用量记录包含租户、项目、应用、Key、模型、供应商、Token、成本、销售价、状态、耗时。
- [ ] 预算超限返回明确错误码。
- [ ] 调用日志可查询。
- [ ] 操作审计可查询。
- [ ] 未完成 V1/V2 功能点击时提示暂不可用。
- [ ] Docker Compose 可一键启动。
- [ ] 交付包包含 NOTICE、第三方声明、开源合规说明。
